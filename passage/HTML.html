HTML简介

什么是HTML?

HTML 是用来描述网页的一种语言,指的是超文本标记语言: HyperText Markup Language。 HTML 不是一种编程语言，而是一种标记语言。 标记语言是一套标记标签 (markup tag)。 HTML 使用标记标签来描述网页，其文档包含了HTML 标签及文本内容 ，HTML文档也叫做 web 页面

HTML 标签

HTML 标记标签通常被称为 HTML 标签 (HTML tag)。 HTML 标签是由尖括号包围的关键词，比如 < html > HTML 标签通常是成对出现的，比如 < b > 和 < /b > 标签对中的第一个标签是开始标签，第二个标签是结束标签 开始和结束标签也被称为开放标签和闭合标签。
